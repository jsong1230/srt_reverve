## 인수조건

### F-01: SRT 로그인
- [x] SRT 회원 ID/비밀번호로 로그인 가능
- [x] 로그인 성공 여부 확인 (환영 메시지, URL 변경, 로그인 폼 사라짐)
- [x] Alert 발생 시 자동 처리
- [x] ElementClickInterceptedException 발생 시 JavaScript 클릭으로 재시도

### F-02: 기차 검색
- [x] 출발역/도착역 입력 및 검증 (17개 역 목록)
- [x] 날짜(YYYYMMDD) 및 시간(짝수 시간) 형식 검증
- [x] 조회 버튼 클릭 후 검색 결과 페이지 표시
- [x] 사용 가능한 날짜 옵션 범위 확인

### F-03: 자동 새로고침 및 예약
- [x] 검색 결과를 반복 확인하며 예약 가능 좌석 탐색
- [x] 15~30초 랜덤 간격으로 새로고침 (봇 탐지 회피)
- [x] 예약 가능 좌석 발견 시 자동 예약 시도
- [x] 예약 성공 시 is_booked 플래그 설정 및 브라우저 창 유지

### F-04: 예약 대기 신청
- [x] --reserve 옵션 활성화 시 예약 대기 신청 시도
- [x] 대기 신청 완료 시 is_booked 플래그 설정

### F-05: 봇 탐지 우회
- [x] undetected-chromedriver 모드: ChromeDriver 바이너리 패치
- [x] selenium-stealth 모드: JavaScript 레벨 자동화 흔적 숨김
- [x] enhanced 모드: 강화된 Chrome 옵션 + JS 스크립트 주입
- [x] 인간 시뮬레이션: 랜덤 대기, 자연스러운 타이핑, 부드러운 스크롤, 마우스 이동

### F-06: CLI 인터페이스
- [x] 필수 인자: --user, --psw, --dpt, --arr, --dt, --tm
- [x] 선택 인자: --num, --reserve, --anti-bot, --delay-min, --delay-max, --use-profile, --profile-dir
- [x] 누락된 필수 인자 감지 및 에러 메시지 출력

### F-07: 설정 관리
- [ ] .env 파일에서 SRT_USER, SRT_PASSWORD, SRT_DPT, SRT_ARR 등 환경변수 로드
- [ ] .env 파일이 없으면 CLI 인자로 폴백
- [ ] .env.example 템플릿 파일 제공
- [ ] CLI 인자가 .env 값보다 우선 적용

### F-08: 에러 리커버리
- [ ] 네트워크 오류 (TimeoutException, ConnectionError) 발생 시 최대 3회 자동 재시도
- [ ] 세션 만료 감지 시 자동 재로그인 후 검색 재개
- [ ] 브라우저 크래시 감지 시 WebDriver 재초기화 후 전체 프로세스 재시작
- [ ] 재시도 횟수 및 복구 이력 로깅
- [ ] 최대 재시도 횟수 초과 시 명확한 에러 메시지와 함께 종료

### F-09: Telegram 알림
- [ ] Telegram Bot Token + Chat ID를 .env 또는 CLI로 설정
- [ ] 예약 성공 시 열차 정보 포함 Telegram 메시지 발송
- [ ] 예약 실패(최대 재시도 초과) 시 실패 알림 발송
- [ ] Telegram 설정이 없으면 알림 기능 비활성화 (에러 없이 무시)
- [ ] 네트워크 오류로 알림 발송 실패 시 예약 프로세스에 영향 없음

### F-10: 로깅 개선
- [ ] logs/ 디렉토리에 날짜별 로그 파일 저장 (예: srt_2026-03-11.log)
- [ ] 로그 파일 자동 회전 (7일 보관)
- [ ] --log-level 옵션으로 로그 레벨 설정 (DEBUG/INFO/WARNING/ERROR)
- [ ] 콘솔 출력과 파일 출력 동시 지원 (듀얼 핸들러)

### F-11: 다중 검색 조건
- [ ] 여러 날짜 입력 가능 (예: --dt 20260315,20260316)
- [ ] 여러 시간대 입력 가능 (예: --tm 08,10,12)
- [ ] 조건을 순차적으로 검색하여 가장 먼저 예약 가능한 좌석 예약
- [ ] 모든 조건을 1회씩 확인 후 다시 처음부터 반복

### F-12: 헤드리스 모드
- [ ] --headless 옵션으로 브라우저 UI 없이 실행
- [ ] 헤드리스 모드에서도 봇 탐지 우회 기능 정상 동작
- [ ] 예약 성공 시 결과를 콘솔 로그로 출력 (브라우저 창 확인 불가하므로)
