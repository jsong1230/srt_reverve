## [v1.0.0] - 2026-03-12

### 🎉 마일스톤 완료

모든 기능 F-01~F-12 완료, 프로덕션 레벨 안정성 달성

### ✨ 주요 기능

#### F-01: 기본 로그인 및 예약
- SRT 사이트 자동 로그인
- 기차 검색 및 예약
- 예약 결과 자동 확인

#### F-02~F-04: 봇 탐지 우회
- `undetected-chromedriver` 지원 (가장 효과적)
- `selenium-stealth` 지원 (중간 난이도 회피)
- `enhanced` 기본 방법 (Chrome 옵션 최적화)

#### F-05~F-07: 설정 및 검증
- `.env` 파일 기반 설정 관리
- 역명, 날짜, 시간 검증
- CLI 인자 파싱 (argparse)

#### F-08: 에러 복구 (Recovery Pattern)
- 네트워크 오류 자동 복구
- 세션 만료 자동 재로그인
- 브라우저 연결 끊김 감지 및 복구

#### F-09: Telegram 알림
- 예약 성공/실패 알림
- 에러 발생 시 즉시 통보
- 구독자 관리

#### F-10: 로깅 개선
- 파일 저장 (날짜별 로그 회전)
- 로그 레벨 설정 옵션 (DEBUG, INFO, WARNING, ERROR)
- 구조화된 로그 포맷

#### F-11: 다중 검색 조건
- 쉼표 구분 다중 날짜 입력 (`--dt 20260315,20260316`)
- 쉼표 구분 다중 시간 입력 (`--tm 08,10,12`)
- 순차 검색으로 예약 확률 증가

#### F-12: 헤드리스 모드
- `--headless true/false` 옵션
- UI 없이 백그라운드 실행
- 서버 환경 지원 (Docker, VPS)

## [v0.5.0] - 2026-02-16

### ✨ 주요 추가

#### 봇 탐지 우회 3가지 방법 추가
- **undetected-chromedriver**: Cloudflare, DataDome 우회 (⭐ 권장)
- **selenium-stealth**: JavaScript 레벨 은폐
- **enhanced**: Chrome 옵션 최적화

#### 인간처럼 동작하는 기능
- 랜덤 대기 (0.5~2초)
- 자연스러운 타이핑 (글자 하나씩 0.05~0.15초)
- 부드러운 스크롤
- 마우스 이동 시뮬레이션

#### JavaScript 스크립트 주입
- `navigator.webdriver` 숨기기
- `window.chrome` 객체 추가
- 권한 정보 변조

### 🐛 버그 수정

- Chrome 옵션 향상 (`--disable-blink-features=AutomationControlled`)
- 타이밍 이슈 해결
- 알림 전송 안정성 개선
